# Recommendation Systems Series

A comprehensive tutorial series on building recommendation systems from scratch, covering fundamental algorithms and practical implementations.

## 📚 Overview

This repository contains hands-on tutorials for understanding and implementing various recommendation system algorithms. Each notebook provides a complete implementation with detailed explanations, making it perfect for learning the fundamentals of recommendation systems.

## 🗂 Repository Structure

```
recsysSeries/
├── 1. Recsys Fundamentals/
│   ├── 1. ContentBased.ipynb          # Content-based filtering implementation
│   ├── 2. Collaborative Filtering.ipynb  # Collaborative filtering methods
└── README.md                          # This file
```

## 📖 Tutorials

### 1. Content-Based Filtering
**File:** `1. Recsys Fundamentals/1. ContentBased.ipynb`

Learn how to build recommendation systems that suggest items based on item features and user preferences.

### 2. Collaborative Filtering
**File:** `1. Recsys Fundamentals/2. Collaborative Filtering.ipynb`

Explore collaborative filtering approaches that make recommendations based on user behavior patterns.
